# Claude Code Usage

Recommended npx install:

```bash
npx webcraft-skills install --agent claude
```

This installs the skill to `~/.claude/skills/webcraft-ui` and command prompts to `~/.claude/commands`.

Then use the installed slash command prompts or explicit prompt-style requests:

```text
/ui-audit Standard Audit for the current website
/ui-fix Critical and Major issues from the last audit
Use webcraft-ui to audit this UI.
```
