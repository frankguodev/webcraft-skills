# AI UI Constitution

English | [中文](./README_zh_CN.md)

Reusable UI prompt presets for Cursor, Codex, Claude Code, v0, Windsurf, Lovable, and other AI coding tools.

AI coding tools can generate code quickly, but their UI often feels too crowded, flashy, inconsistent, or obviously AI-generated.

---

# What Is This?

AI UI Constitution is a lightweight prompt rules library.

It is not a UI framework, design system, component library, or product builder.

The goal is simple:

- cleaner layouts
- better spacing
- calmer motion
- more consistent UI
- stronger typography
- less AI-template feeling

Copy a preset, paste it into your AI coding tool, then describe the page you want.

---

# Quick Start

1. Open a preset from [en/presets](./en/presets/).
2. Fill in the project or personal information section when the preset provides one.
3. Copy the prompt rules.
4. Paste them into your AI coding tool.
5. Add your page request and generate the UI.
