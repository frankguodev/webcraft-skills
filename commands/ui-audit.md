# /ui-audit

Run a strict visual, responsive, interaction, and AI-template-smell audit.

## Usage

```text
/ui-audit [scope]
```

Examples:

```text
/ui-audit whole site
/ui-audit current localhost app
```

## Agent Instructions

Use the `ai-ui-constitution` skill in Audit mode. Inspect layout, typography, color, borders, radius, shadow, components, modals, responsive behavior, motion, accessibility, and AI template smell. Prefer browser verification when available.
