# existing-style-plus-design-ref

Purpose: test the build rule for reference images/design images.

Scenario:

```text
在这个现有项目里，根据 reference-layout.txt 新增一个 homepage section。
这张图只作为布局结构参考，风格沿用现有网站。
```

Expected behavior:

- Use the existing light, quiet product style in `index.html`.
- Use the reference only for structure and information order.
- Do not switch to the reference's dark editorial visual style.
